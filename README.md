# BFS 广度优先搜索演示项目

一个交互式的广度优先搜索 (Breadth-First Search) 算法演示项目，包含 Python 算法实现和浏览器端可视化动画。

## 项目结构

```
try-to-git/
├── bfs/                   # Python BFS 算法包
│   ├── __init__.py        # 包导出
│   ├── graph.py           # 图数据结构 & BFS 核心实现
│   ├── maze.py            # 迷宫生成器（递归回溯法）
│   └── demos.py           # 4 个应用场景演示
├── main.py                # Python 入口文件
├── index.html             # 交互式可视化页面（浏览器直接打开）
└── README.md
```

## 功能特性

### Python 算法实现
- **Graph 类** — 基于邻接表的图，支持有向/无向图
- **bfs()** — 标准 BFS 遍历，返回访问顺序、层级、父节点
- **bfs_shortest_path()** — BFS 最短路径查找

### 5 个演示场景（命令行）
1. **社交网络** — 好友关系遍历与最短连接路径
2. **地铁线路** — 站点间最少站数路径
3. **迷宫求解** — 随机迷宫 BFS 寻路
4. **二叉树层序遍历** — BFS 经典应用
5. **单词接龙** — 单词间最短转换路径（每次只能替换一个字母）

### 交互式可视化（浏览器）
- **图遍历** — 可视化 BFS 逐层扩展过程，支持自动/单步执行
- **最短路径** — 动画展示 BFS 如何找到最短路径
- **迷宫求解** — 随机迷宫生成 + BFS 实时搜索动画，支持 3 种尺寸
- **单词接龙** — 可视化单词转换图，动画展示 BFS 搜索最短路径
- 算法原理说明与复杂度分析

## 快速开始

### 运行 Python 演示
```bash
python main.py
```

### 打开可视化页面
直接在浏览器中打开 `index.html`，无需安装任何依赖。

## 算法复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(V + E) | V=顶点数, E=边数 |
| 空间 | O(V) | 队列和 visited 集合 |
| 最短路径 | 保证 | 在无权图中一定找到最短路径 |
